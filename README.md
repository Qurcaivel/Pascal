Структура проекта:

- Core - главный модуль проекта, отвечающий за делегирование 
этапов интерпретации другим модулям.

- Lex - модуль, отвечающий за лексический анализ
исходного кода и генерации списка токенов.

- AST - модуль, содержащий в себе структуры, необходимые
для представления исходного кода в виде абстрактного
синтаксического дерева.

- Parse - модуль, отвечающий за синтаксический анализ
исходного кода и преобразования списка токенов
в абстрактное синтаксическое дерево.

- Venv - модуль, отвечающий за исполнение конечного
представления исходного кода программы.

- Test - модуль, необходимый для тестирования основных
модулей проекта с различными входными данными.

- Optimize - модуль, необходимый для оптимизации абстрактного
синтаксического дерева.

- Resolver - модуль, отвечающий за проверку достижимости имен,
а также допустимости типа выражений.

- Debug - модуль, содержащий элементы для вывода отладочной 
информации на экран.
